About Me

Hai , saya Regina Kartika Sari

Saya adalah seorang fresh graduate, saat ini saya berusia 18 tahun, saya berasal dari Blitar, Jawa Timur. Asal sekolah saya dari SMK Brantas Karangkates Malang, disana saya mengambil jurusan Rekayasa Perangkat Lunak. Saat ini saya sedang mengikuti program pendidikan UT School IT, alasan saya mengikuti program tersebut karena saya ingin memperdalam apa yang telah saya pelajari semasa SMK, yaitu pemrograman.

Pada saat itu saya PKL di PT Universal Big Data, yang merupakan sebuah perusahaan start up disana saya menjadi tim membuat konten website dengan menggunakan WordPress selain itu disana saya juga membuat video explainer, dan membuat desain poster.
Selama kegiatan pembelajaran di sekolah, project yang pernah dikerjakan yaitu membuat CRUD menggunakan Codeigniter, membuat aplikasi mobile dengan Kodular, serta membuat aplikasi pengelola data menggunakan platform Serenity.
